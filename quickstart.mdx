Getting started with seeksphere is pretty simple just follow the steps below after you have create an account and setup an org:

- Fill the [Setup form](/onboarding/setupForm)
- Verify the [Data Schema](/how-seeksphere-works/dataSchema) in the Data Schema Page
- Create a CDC Pipeline on your infra, use the following docs for AWS, GCP and native function.
- Integrate the seeksphere into your application using SDKs or APIs.
